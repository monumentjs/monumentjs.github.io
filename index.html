@monument/core

Install
npm install @monument/core
Collections

This packages provides useful classes and interfaces to work with collections such as List, Set, Queue, Stack, Map and more specialized.

For each type of collection there are read-only interface (which allows only those operation which don't mutate collections).

Each type of collection also have mutable and immutable implementations.

Mutable collections are also observable which means you can subscribe on updates.

Events

This package provides useful classes and interfaces for defining typed events as object properties.

Implementation follows "event as property" approach.

Exceptions

This package provides base exceptions.

Comparison

This package provides abstraction of equality comparison and sorting.

String manipulation

This package provides additional classes to work with text like string builder, parsers, template string and others.
